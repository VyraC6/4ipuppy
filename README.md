# 4iPuppy 🐶

为4i情侣打造的互动卡牌小游戏。纯前端实现，可直接部署到 GitHub Pages。

## 玩法

1. 选择热度模式：轻松甜蜜 / 慢慢升温 / 火热模式
2. 点击卡牌翻牌，查看挑战/问题
3. 完成后点击"完成"，不想做点击"跳过"
4. 翻完全部卡牌后查看总结

## 卡片类别

- 🎭 角色反转 — 关于性别角色的温柔反叛
- 💬 真心话 — 直抵心底的诚实时刻
- ⚡ 互动挑战 — 放下手机，用身体来证明默契
- 🔥 升温时刻 — 越过安全区（仅温/热模式）

## 部署到 GitHub Pages

1. Fork 或上传此仓库到 GitHub
2. 在仓库 Settings → Pages 中，选择 main 分支作为源
3. 保存后即可通过 `https://你的用户名.github.io/仓库名` 访问

## 本地运行

```bash
# 用任意静态服务器即可
python3 -m http.server 8080
# 然后打开 http://localhost:8080
```

## 技术

纯 HTML + CSS + JavaScript，无框架依赖。
